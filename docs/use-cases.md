# Use Cases

This project will evaluate Xiaomi MiMo API in Japanese-language AI workflows.

## 1. OpenClaw-Compatible Agent Backend

MiMo API will be tested as a backend model for OpenClaw-like AI agent workflows.

Evaluation points:

- OpenAI-compatible API behavior
- response quality in Japanese
- coding assistance
- tool-use style prompting
- workflow automation

## 2. Japanese Coding Assistant

MiMo models will be tested for Japanese developer workflows, including:

- TypeScript code generation
- debugging assistance
- README generation
- command-line setup guidance
- error explanation

## 3. Long-Context Document Tasks

The project will test document summarization and reasoning tasks in Japanese.

Example tasks:

- summarizing university materials
- extracting requirements from documents
- comparing multiple sources
- generating structured reports

## 4. Student Study Assistant

MiMo API will be tested for Japanese university study support.

Example tasks:

- lecture note summarization
- English and Chinese translation support
- report drafting support
- exam preparation prompts

## 5. Public Documentation

The project will publish:

- setup guide
- API examples
- OpenClaw configuration notes
- benchmark prompts
- Japanese-language usage notes
