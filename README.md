# MiMo OpenClaw Agent Harness

This repository is a lightweight integration harness for testing Xiaomi MiMo API with OpenClaw, Cursor-style coding workflows, and OpenAI-compatible agent tools.

## Project Goal

The goal is to evaluate MiMo-V2.5 models in Japanese-language AI workflows, including:

- coding assistance
- Japanese reasoning
- long-context document analysis
- university study support
- AI agent task execution
- personal workflow automation

## Why MiMo

Xiaomi MiMo API provides access to MiMo-V2.5 models and is suitable for testing agentic workflows, coding tools, and Japanese-language AI applications.

This project will document how to connect MiMo API to OpenClaw-like and Cursor-like development workflows.

## Planned Features

- MiMo API call examples
- OpenAI-compatible API wrapper examples
- OpenClaw configuration notes
- Cursor-style coding assistant tests
- Japanese reasoning benchmark prompts
- document summarization tests
- setup guide for Japanese developers

## Target Users

This project is intended for:

- Japanese AI developers
- students experimenting with AI agents
- developers evaluating MiMo API
- users building OpenClaw-compatible AI workflows

## Current Status

Prototype stage.

The repository will be updated with:

- setup instructions
- API usage examples
- benchmark prompts
- Japanese documentation
- comparison notes with other AI coding workflows

## Safety

No API keys or private credentials will be committed to this repository.
